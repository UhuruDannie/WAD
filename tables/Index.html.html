Web Application Development

Course code:ITM 1131

Course level:Year one Semester one

Credit Unit:4

Course Description

Thiscourse introduces students to web technologies-HTML,CSS,Javascript;CSS frameworks ;javascript frameworks.students will learn to implement modern web pages using HTML and CSS frameworks.The course also introduces students to full stack web application progamming using Javascript.

Course Objectives

  1. Describe web technologies(HTML,CSS &Javascript),CSS trameworks,and Javascript frameworks.
  2. Implement a simple but modern web page with HTML and CSS.
  3. Build a simple but functional web application using Javascript.

Learning Outcomes

  • Use web technologies(HTML and CSS frameworks.
  • Design and create a simple but functional web page using HTML and CSS frameworks.
  • Compose and build a simple full stack web application using Javascript frameworks.
